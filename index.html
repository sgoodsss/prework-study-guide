HTML

  • HTML is used to create the structure of a webpage.
  • HTML is made up of elements. Elements are used to add content.
  • HTML attritbutes define additional information about an element.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Parent Element- element that holds the children elements
  • Child Elements- Elements that are nested inside other elements

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud based hosting service that allows us to keep track of our projects inside repositories.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: Adds all modifications in the current working branch to staging area.
  • git commit: commits the changes
  • git pull origin main: checks if our local branch is in sync with the main branch in GitHub.
  • git push origin branch-name: pushes changes to the remote branch.
