HTML

  • HTML is used to create the structure of a webpage.
  • HTML is made up of elements. Elements are used to add content.
  • HTML attritbutes define additional information about an element.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Parent Element- element that holds the children elements
  • Child Elements- Elements that are nested inside other elements

CSS

  • CSS is used to style a webpage.
  • 3 ways to style with CSS- Inline CSS, Internal CSS Stylesheet, External CSS Stylesheet.
  • This project uses an external css stylesheet, which means it is linked through the head of our HTML page.
  • Declarations contain the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud based hosting service that allows us to keep track of our projects inside repositories.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: Adds all modifications in the current working branch to staging area.
  • git commit: commits the changes
  • git pull origin main: checks if our local branch is in sync with the main branch in GitHub.
  • git push origin branch-name: pushes changes to the remote branch.

JavaScript

  • JavaScript allows users to interact with websites.
  • JavaScript is linked under the footer and above the closing element of our HTML page.
  • A variable is a named container that allows us to store data in our code.
  • console.log method is used to output a message to the web console by adding an argument.
  • Control flow is the order in which a computer executes code in a script.
  • If Statements: check whether a condition evaluates to be true or false. Answer determines the next course of action for our app.
  • Strict Equality Operator (===): Checks to see if 2 values are equal.
  • Array: a single variable that is used to hold a group of data.
  • Index: the location of a data item in an array, located with a unique number. First item in an index is always 0.
  • for loop: uses the pattern of indicies to perform a taks on all the items in an array by allowing a single block of code to be executed over and over.
  • Function: a set of instructions that tells the computer how to perform a certain task.
  • 2 requirements for a JS function: 1. Define the function, 2. Call the function.
